## 음원 제작 기준

- MP3, 44.1kHz 또는 48kHz
- 효과음은 가능하면 모노, 음악은 스테레오
- 시작과 끝의 불필요한 무음 제거
- 효과음 최대 피크: 약 `-1 dB`
- 평균 음량 권장: 효과음 `-16~-12 LUFS`, 음악 `-24~-20 LUFS`
- 배경음은 끝과 시작이 튀지 않는 완전한 루프
- 파일명은 영문 소문자와 밑줄만 사용
- 저작권과 상업적 게임 사용 권한을 확인한 음원만 사용

기존 `assets/sounds/*.wav`는 MP3 세트가 완성될 때까지 안전한 대체음으로 유지합니다.
